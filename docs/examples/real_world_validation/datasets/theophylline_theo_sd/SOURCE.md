# theo_sd provenance

This dataset is the `theo_sd` dataset distributed by the R package `nlmixr2data`.

Description:
- Day 1 concentrations from the theophylline data used in nlme/NONMEM teaching examples.
- Columns include: ID, TIME (hr), DV (mg/L), AMT (mg), EVID, CMT, WT (kg).
