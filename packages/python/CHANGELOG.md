# Changelog

All notable changes to OpenPKPD are documented here.

## [0.1.0] Initial public infrastructure release

### Core
- Deterministic PK and PKPD simulation
- Explicit event and solver semantics
- Population simulation with IIV, IOV, covariates, and time-varying covariates
- Sensitivity analysis with replayable artifacts

### Reproducibility
- Artifact-based execution
- Golden artifact validation
- Semantics fingerprint enforcement

### Tooling
- CLI for replay and validation
- Python bindings via juliacall
- Executable documentation contracts
